# watime
A current time formatted written in Rust.

Get the current time.
```
watime
```
Defaults to millis
```
1651153473890
```

## Formats
### Seconds
Accepts `s` or `second` or `seconds`
```
watime -f s
```

### Millis
Accepts `m` or `ms` or `milli` or `millis`
```
watime -f m
```

### Nanos
Accepts `n` or `ns` or `nano` or `nanoss`
```
watime -f n
```

### ISO RFC3339
```
watime -f iso
```

## Help
```
watime -h
```
